Cubic Function

Transformations

Parent Function: f(x) = x³ + x² - 7x
Transformed: g(x) = 0.158 * f( 0.598(x + 0.89) ) + 0.47

The parent cubic function, f(x) = x³ + x² - 7x, was transformed to model the motion of a roller coaster in Disney Park. The image shows a section of the ride’s shape featuring an uphill followed by a steep downhill, representing the thrill and motion of the coaster.

This specific parent function was chosen precisely because its natural shape already includes this "uphill-then-downhill" curve. The transformations were then applied to scale and position this shape onto the real track.

It was vertically compressed by a factor of 0.158 (a = 0.158). This positive 'a' value reduces the steepness of the parent function's curve to reflect the gentler slope visible in the real track. A horizontal stretch by a factor of 1.672 (k ≈ 0.598) widens the curve, matching the long, smooth transition.

Finally, the function was translated 0.89 units to the left (d = -0.89) and 0.47 units upward (c = 0.47), aligning the function's peak with the coaster’s base level. Together, these transformations allow the cubic function to realistically represent the roller coaster’s shape.

Domain and Range

The domain of this polynomial function is approximately -6 < x < -1.7, representing the section of the roller coaster track shown in the image — from the point where the train begins to climb the hill to where it finishes descending.

The range reflects the vertical positions of the coaster along this segment of the track. Over the given domain, the function rises from a low point of approximately 0.81 to a maximum height near 2.04. Therefore, the range is 0.81 < y < 2.04, capturing the full vertical height of this portion of the ride.

Support Frame of the Pirate Ship Ride

Absolute Function

Transformations

Parent Function: f(x) = |x|
Transformed Function: g(x) = -1.3|x - 1| + 3.3

The parent absolute value function was transformed to model the support frame of the pirate ship ride, whose structure forms a tall, symmetrical “V” shape supporting the swinging motion of the ship. The graph is reflected vertically because the leading coefficient is negative (a = -1.3), flipping the parent function downward to resemble the beams hanging from the central pivot.

It is vertically stretched by a factor of 1.3, making the “V” shape steeper and more representative of the long, angled metal beams that rise from the ground to hold the ship. The graph is then translated 1 unit to the right (d = 1), positioning the vertex directly beneath the central pivot of the ride, and shifted 3.3 units upward (c = 3.3) so the vertex sits at its correct height, consistent with the actual structure.

Together, these transformations allow the absolute value function to accurately model the geometric symmetry, slope, and position of the pirate ship’s support frame in real life.

Domain and Range

The domain of this absolute value function is -4 < x < 5, representing the horizontal span of the two supporting beams of the structure. This interval captures the full width of the frame from one end to the other along the ground.The range reflects the vertical positions of the frame. The function reaches its maximum height of 3.3 at the pivot point where the beams meet the top of the structure. The minimum height occurs at -3.2, corresponding to the base of the beam where it is anchored to the ground, supporting the entire structure. Therefore, the range is -3.2 ≤ y ≤ 3.3, representing the full vertical extent of the frame from the ground-based supports to the high pivot point.

Transformations

Parent Function: f(x) = sin(x)
Transformed Function: g(x) = 1.5sin(0.45(x + 2.8)) - 0.5

The parent sine function was transformed to model the top section of a roller coaster track, where the curve reaches its highest peak before descending. The graph is vertically stretched by a factor of 1.5 (a = 1.5), making the wave taller and emphasizing the rise and fall of the coaster’s motion. It is also horizontally stretched by a factor of about 2.22 (k = 0.45), which lengthens the curve to match the wider, gentler slope of the coaster’s upper track.

The function is shifted 2.8 units to the left (d = -2.8), aligning the highest point near the center of the structure, and moved downward by 0.5 units (c = -0.5) to set the center-line of the wave. Together, these transformations create a smooth, elevated sine wave that represents the top arc of the roller coaster.

Domain and Range

The domain of this sinusoidal function is -5 < x < 5, representing the horizontal distance covered by the top portion of the roller coaster shape depicted in the image. This interval captures the width of this segment from one side of the wave crest to the other, rather than a complete full wave.

The range reflects the vertical positions along the coaster’s upper path. Within the restricted domain, the expression 0.45(x+2.8) spans approximately −0.99 to 3.51 radians, over which the sine function reaches a maximum of 1 and a minimum of about −0.836. After applying the vertical stretch of 1.5 and the downward shift of 0.5, the transformed function reaches a maximum height of 1 and a minimum of approximately −1.76, giving a range of −1.76 < g(x) < 1. This range corresponds precisely to the full vertical height of the coaster’s top segment, from the highest point of the shape down to its lower edge, as seen in the image.
