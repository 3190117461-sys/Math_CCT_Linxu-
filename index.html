Roller Coaster Function

Cubic Function

Transformations

Parent Function: f(x) = x³
Transformed Function: g(x) = -0.158[ (1/1.672)(x + 0.89) ]³ + 0.47

The parent cubic function was transformed to model the motion of a roller coaster. Because the leading coefficient (a = -0.158) is negative, the graph is reflected vertically, symbolizing the initial drop after the peak.

It was vertically compressed by a factor of 0.158, reducing the steepness. A horizontal stretch by a factor of 1.672 (k = 1/1.672) widens the curve, matching the long, smooth transition of the coaster’s motion. The function was also translated 0.89 units to the left (d = -0.89) and 0.47 units upward (c = 0.47), which sets the inflection point of the curve.

Domain and Range

For the section of the track shown in the graph, the domain and range are restricted:

Domain: { x ∈ ℝ | -6 ≤ x ≤ -1.7 }
Range: { y ∈ ℝ | 0.5 ≤ y ≤ 5.0 }

Support Frame of the Pirate Ship Ride

Absolute Function

Transformations

Parent Function: f(x) = |x|
Transformed Function: g(x) = -1.3|x - 1| + 3.3

The parent absolute value function models the "V" shape of the pirate ship's support frame. The graph is reflected vertically (a = -1.3) to open downward, representing the two beams meeting at the base.

It is vertically stretched by a factor of 1.3, making the “V” shape steeper to match the angled beams. The graph is then translated 1 unit to the right (d = 1), positioning the vertex at the structure's center, and shifted 3.3 units upward (c = 3.3) to set the vertex's height.

Domain and Range

The physical constraints of the support beams define the restricted domain and range:

Domain: { x ∈ ℝ | -4 ≤ x ≤ 5 }
Range: { y ∈ ℝ | -3.2 ≤ y ≤ 3.3 }

Sinusoidal Function

Transformations

Parent Function: f(x) = sin(x)
Transformed Function: g(x) = 1.5sin(0.45(x + 2.8)) - 0.5

The parent sine function models the periodic wave of the track's hills. It is vertically stretched by a factor of 1.5 (a = 1.5), creating the amplitude of the hill. It is also horizontally stretched by a factor of about 2.22 (k = 0.45), which lengthens the curve.

The function is phase-shifted 2.8 units to the left (d = -2.8) and translated 0.5 units downward (c = -0.5), which establishes the axis of the wave at y = -0.5.

Domain and Range

While the function is periodic, the domain is restricted to the section of the track shown. The range is determined by the amplitude and vertical shift.

Domain: { x ∈ ℝ | -5 ≤ x ≤ 5 }
Range: { y ∈ ℝ | -2 ≤ y ≤ 1 }

(Range is calculated from: Max = c + a = -0.5 + 1.5 = 1. Min = c - a = -0.5 - 1.5 = -2.)

Exponential Function

Transformations

Parent Function: f(x) = 2ˣ
Transformed Function: g(x) = -0.2 • 2⁰.³(ˣ + ¹⁸.¹) + 6.9

The parent exponential function $f(x) = 2^x$ was transformed. The graph is vertically reflected (a = -0.2) and compressed by a factor of 0.2, which flips the curve downward to model the drop.

A horizontal stretch by a factor of approximately 3.33 (k = 0.3) widens the curve. The graph is shifted 18.1 units to the left (d = -18.1) and translated 6.9 units upward (c = 6.9) to position the curve and its horizontal asymptote (y = 6.9).

Domain and Range

The domain and range for this specific piece of the track are:

Domain: { x ∈ ℝ | -2 ≤ x ≤ 0.8 }
Range: { y ∈ ℝ | -3.24 ≤ y ≤ 1.24 }

Square Root Function

Transformations

Parent Function: f(x) = √x
Transformed Function: g(x) = 2.3√(-1.3(x - 5.2)) - 6.3

The parent square root function $f(x) = \sqrt{x}$ models the descending curve. It is vertically stretched by a factor of 2.3 (a = 2.3), making the descent steeper.

The negative horizontal factor (k = -1.3) reflects the graph across the y-axis and compresses it horizontally, making the function decrease from right to left. The function is shifted 5.2 units to the right (d = 5.2) and 6.3 units downward (c = -6.3) to set the starting point of the curve at (5.2, -6.3).

Domain and Range

The restricted domain and range for this section of the track are:

Domain: { x ∈ ℝ | -5 ≤ x ≤ 4.2 }
Range: { y ∈ ℝ | -3.68 ≤ y ≤ 2.08 }
