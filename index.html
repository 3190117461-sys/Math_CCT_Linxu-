Roller Coaster Function

Cubic Function

Transformations

Parent Function: f(x) = x³ + x² - 7x
Transformed: g(x) = 0.158 * f( 0.598(x + 0.89) ) + 0.47

Full g(x) = 0.158 • ( [0.598(x+0.89)]³ + [0.598(x+0.89)]² - 7[0.598(x+0.89)] ) + 0.47

The parent cubic function, f(x) = x³ + x² - 7x, was transformed to model a section of a roller coaster track featuring an uphill followed by a steep downhill. The transformations adjust the curve to fit the real ride in scale and position.

Vertical compression: a = 0.158 reduces the steepness.
Horizontal stretch: k ≈ 0.598 widens the curve.
Horizontal shift: d = -0.89 and vertical shift: c = 0.47 align the peak with the coaster’s base level.

Derivative Analysis (Level 4)

To find the most thrilling point (steepest descent), we calculate the derivative g'(x):
g'(x) = 0.158 * 0.598 * [ 3*(0.598(x+0.89))² + 2*(0.598(x+0.89)) - 7 ]

Within the domain -6 ≤ x ≤ -1.7, we look for the minimum of g'(x) since the steepest downhill corresponds to the largest negative slope. Although mathematically the global steepest point occurs at x ≈ -1.45, this lies outside our modeled interval. Therefore, the actual "most thrilling" point for this section is at the right endpoint x = -1.7.

Domain and Range

Domain: -6 ≤ x ≤ -1.7 (from the start of the climb to the end of the modeled descent).
Range: 0.82 ≤ y ≤ 2.06, representing vertical positions along this segment.

Support Frame of the Pirate Ship Ride

Absolute Function

Transformations

Parent Function: f(x) = |x|
Transformed Function: g(x) = -1.3|x - 1| + 3.3

Models the support frame of the pirate ship ride. Vertical reflection (a = -1.3) flips it downward, vertical stretch increases slope, and shifts (d = 1, c = 3.3) align the vertex with the pivot point.

Domain and Range

Domain: -4 ≤ x ≤ 5, Range: -3.2 ≤ y ≤ 3.3
